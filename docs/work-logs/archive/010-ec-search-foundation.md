## 数学背景

**核心方程**：对本原勾股数 (p,q,r) 和点 P=(kp/r, kq/r)，令 dA=k，引入坡度参数 t 将 dB 有理化：

```
k(t) = 2(t + p/r) / (1 - t²)
dB(t) = (1 + t² + 2tp/r) / (1 - t²)
```

代入 dD 的表达式，令 E = r·dD·(1-t²)，得到超椭圆曲线：

```
E² = F(t)  其中
F(t) = r²t⁴ + 4qrt³ + (2r²+4pq)t² + 4r(2p-q)t + (4p²-4pq+r²)
系数：c4=r², c3=4qr, c2=2r²+4pq, c1=4r(2p-q), c0=4p²-4pq+r²
```

该曲线亏格为 1（通过判别式 -16p²q²<0 确认），是真正的椭圆曲线。

**四顶点检验**：由平行四边形恒等式 dA²+dC²=dB²+dD²，若 dA, dB, dD 均有理，则 dC 有理当且仅当 dB²+dD²-dA² 是完全有理平方。此检验在 `make_point()` 内通过 `rational_sqrt()` 一次完成。

**弦切法余商二次型**：

切线步（在 (t₀,E₀) 处，λ=F'(t₀)/(2E₀)）：
```
A = c4,  B = c3 + 2c4·t₀,  C = c2 - λ² + 2c3·t₀ + 3c4·t₀²
```

割线步（过 (t₁,E₁) 和 (t₂,E₂)，μ=(E₂-E₁)/(t₂-t₁)）：
```
A = c4,  B = c3 + c4(t₁+t₂),  C = c2 - μ² + B(t₁+t₂) - c4·t₁·t₂
```

新 t 值为 h(t)=At²+Bt+C 的根；当判别式 B²-4AC 是完全有理平方时，得到新有理点。

## 关键设计决策

1. **纯 Fraction 算术**：不引入新依赖（cypari2/PARI/GP）。代价是分母快速增长；通过 `max_denom` 参数限制。

2. **两条曲线分支**：对每个种子同时使用 (t, +E) 和 (t, -E)，增加弦切法找到新有理点的机会。

3. **轨道展开策略**：先对所有队列点做切线步，再对当前边界的所有点对做割线步（含跨分支），循环 max_steps 轮。

4. **与第一层搜索分离**：`ec_search` 是独立的 CLI 和模块，不修改现有 `search.py` / `search_gpu.py` 的任何代码。

## 局限与后续方向

- **轨道展开稀疏**：对小三元组（如 (3,4,5)），弦切步的判别式多为非完全平方，轨道不展开。需要更多种子或更大的种子搜索范围。
- **分母增长**：每一步切线/割线后分母平方增长，几步后即达 `max_denom` 限制（默认 10¹⁸）。实用时需根据机器精度权衡。
- **未使用 EC rank**：目前不计算椭圆曲线的 Mordell-Weil 群秩。秩=0 的曲线只有有限个有理点；秩>0 才有无穷个。引入 PARI/GP 或 sage 可自动计算秩，从而跳过秩=0 的三元组。
- **四顶点解**：理论上若某个轨道点使 dC 也有理，即找到四顶点解（Harborth 猜想的解）。但目前在 scale=400 范围内未找到。
