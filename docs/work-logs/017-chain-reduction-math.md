# 017 — Chain 搜索的数学化简：从四元组到二元组

## 背景

本文记录将单位正方形四顶点有理距离问题（通过 Pythagorean 4-cycle 建模）化简为
**二参数搜索**的完整推导过程。原始推导由用户提出关键约束，AI 完成代数化简。

---

## 一、问题设置

设四个正整数 $a, b, c, d$（互不相同，且 $ac \neq bd$），满足：

$$a^2+b^2 = x_1^2, \quad b^2+c^2 = x_2^2, \quad c^2+d^2 = x_3^2, \quad d^2+a^2 = x_4^2$$

以及单位正方形约束：

$$a + c = b + d$$

设每个勾股对由**本源勾股三元组** $T_i = (p_i, q_i, h_i)$（$p_i^2+q_i^2=h_i^2$，$\gcd(p_i,q_i)=1$）
和缩放因子 $k_i$ 给出：

$$a = k_1 p_1, \quad b = k_1 q_1 = k_2 p_2, \quad c = k_2 q_2 = k_3 p_3, \quad d = k_3 q_3 = k_4 p_4, \quad a = k_4 q_4$$

---

## 二、初步约束推导

### 2.1 必须用四个不同的本源三元组

从 $a + c = b + d$ 出发，写成各 $k_i$ 的形式：

$$k_1(p_1 - q_1) + k_3(p_3 - q_3) = 0 \qquad \text{(I)}$$

因为 $p_i \neq q_i$（否则 $h_i^2 = 2p_i^2$ 不是完全平方），所以对角的 $T_1, T_3$ 必须满足
$(p_1-q_1)$ 与 $(p_3-q_3)$ **异号**（否则等式无正整数解）。同理 $T_2, T_4$ 的 $p_i-q_i$ 异号。

**结论**：对角的本源三元组不同，且它们的"差 $p-q$"符号相反。

### 2.2 $k_i$ 的链式关系

由相邻项共享同一整数（$b = k_1 q_1 = k_2 p_2$ 等）得：

$$k_2 = \frac{k_1 q_1}{p_2}, \quad k_3 = \frac{k_1 q_1 q_2}{p_2 p_3}, \quad k_4 = \frac{k_1 q_1 q_2 q_3}{p_2 p_3 p_4}$$

### 2.3 闭合条件

链首尾相连 $k_4 q_4 = k_1 p_1$ 给出：

$$\boxed{q_1 q_2 q_3 q_4 = p_1 p_2 p_3 p_4} \qquad (*)$$

---

## 三、关键化简：消去所有 $k_i$

将 $k_2, k_3$ 代入 (I) 并约掉 $k_1$，得**纯本源腿方程**：

$$\boxed{p_2 p_3 (p_1 - q_1) + q_1 q_2 (p_3 - q_3) = 0} \qquad \text{(A)}$$

整理得 $T_3$ 的腿之比由 $T_1, T_2$ 完全决定：

$$\frac{p_3}{q_3} = \frac{q_1 q_2}{N}, \qquad \text{其中} \quad N := p_2(p_1 - q_1) + q_1 q_2$$

由闭合条件 $(*)$ 同理可得 $T_4$ 的腿之比：

$$\frac{p_4}{q_4} = \frac{N}{p_1 p_2}$$

**即：给定 $T_1, T_2$，$T_3$ 和 $T_4$ 的形态完全确定。**

---

## 四、$N$ 的结构

设 $A := q_1 q_2$（小腿之积），$B := p_1 p_2$（大腿之积），约定 $p_i > q_i$，则：

$$N - A = p_2(p_1 - q_1) > 0$$
$$B - N = q_1(p_2 - q_2) > 0$$

因此 $A < N < B$，即 $N$ 严格夹在 $A$ 和 $B$ 之间。

---

## 五、两个完全平方条件（核心约束）

$T_3$ 和 $T_4$ 是合法勾股对，等价于：

$$\boxed{A^2 + N^2 \text{ 是完全平方数}} \qquad (C_3)$$
$$\boxed{N^2 + B^2 \text{ 是完全平方数}} \qquad (C_4)$$

设 $g_3 = \gcd(A, N)$，$g_4 = \gcd(N, B)$，则：

$$T_3 = \left(\frac{A}{g_3},\ \frac{N}{g_3},\ \frac{\sqrt{A^2+N^2}}{g_3}\right), \quad
  T_4 = \left(\frac{N}{g_4},\ \frac{B}{g_4},\ \frac{\sqrt{N^2+B^2}}{g_4}\right)$$

（验证两者本源性：$\gcd(A/g_3, N/g_3)=1$ 等需另行检查。）

---

## 六、异号条件自动成立

$p_1 > q_1$ 且 $p_2 > q_2$ 时，$N > A$ 故 $p_3 < q_3$（即 $T_3$ 的"小腿"更大），
$N < B$ 故 $p_4 < q_4$ iff $N < p_1 p_2$（自然成立）。  
因此方程 (I) 所要求的对角异号条件由公式**自动满足**，无需额外筛选。

---

## 七、积恒等式

由 $(C_3)$ 和 $(C_4)$，利用 Brahmagupta–Fibonacci 恒等式展开：

$$(h_3 h_4)^2 = (A^2+N^2)(B^2+N^2) = (AB + N^2)^2 + N^2(B-A)^2$$

即 $(AB+N^2,\ N(B-A),\ h_3 h_4)$ 本身是一个（可能非本源的）勾股三元组。

另有差值关系：

$$h_3^2 - h_4^2 = A^2 - B^2 = (q_1 q_2 - p_1 p_2)(q_1 q_2 + p_1 p_2)$$

---

## 八、算法意义：从 O(n⁴) 降至 O(n²)

原 chain 搜索枚举所有四元组，等价于 O(n⁴)。
基于以上推导，新算法只需枚举**本源三元组对** $(T_1, T_2)$：

1. 生成所有本源三元组 $T = (p, q, h)$，$p > q$，$h \leq M$
2. 对每对 $(T_1, T_2)$，计算 $N = p_2(p_1-q_1)+q_1 q_2$
3. 检查 $A^2+N^2$ 和 $N^2+B^2$ 是否为完全平方
4. 通过即得 $T_3, T_4$，再还原 $a, b, c, d$

由于本源三元组在 $h \leq M$ 范围内约有 $O(M)$ 个（密度 $\sim M/(2\pi)$），
搜索复杂度降至 $O(M^2)$ vs 原来的 $O(M^4)$，**差约四个数量级**。

---

## 九、尚待研究的方向

1. **$N$ 作为公共腿的数论条件**：$N$ 同时出现在与 $A$ 和 $B$ 的勾股对中，这是一个强约束，
   可能可以通过 Gaussian 整数分解进一步刻画。

2. **$A, B, N$ 的整除性**：$N-A = p_2(p_1-q_1)$ 和 $B-N = q_1(p_2-q_2)$ 说明
   $N-A$ 含因子 $p_2$，$B-N$ 含因子 $q_1$，可用于加速筛选。

3. **对称性**：交换 $T_1 \leftrightarrow T_2$ 得到不同的 $N' = p_1(p_2-q_2)+q_1 q_2$，
   满足 $N + N' = 2B - (p_1 q_2 + p_2 q_1)$，两个方向的搜索可同时进行。

4. **是否存在解**：目前尚无证明四个条件（$A^2+N^2$ 和 $N^2+B^2$ 均为完全平方，
   且 $T_3, T_4$ 本源）同时成立的三元组对。
