# wl102 — 非互素全空间扫描（阶段1–2）：含非互素的"无反例到界"声明

执行 wl101 规划：生成器去掉互素/even-even 过滤（覆盖非互素半空间），按 sound 三段管线
`gcd_aware_kills → chain_closure 模 p² → GEN-CLOSURE` 判定每个 multi-N 对。
脚本 `scripts/multi_n/noncoprime_full_scan.py`。

- `gen_complete_set_mismatch=0`：`exact_concordant_pair` 重算的完整 concordant 集与生成器一致，判定器对非互素精确。

  此前解析筛线只覆盖互素，非互素只有 hyp≤2000 一窗，现已推到 hyp≤100k（非互素对从 1.8k → 31.5 万，175×）。

## 残余结构（与 wl100 一致）

100k 处 GEN-CLOSURE 残余 19,219，gcd 分布仍 `12∣g` 主导：g=12 (1831)、60 (1256)、24 (1115)、120 (674)、36 (646)…；
另有少量互素残余（g=1: 9、g=2: 7）——即满足 $12\mid(A+B)$、模筛杀不掉、GEN-CLOSURE 确认无闭合的互素对。
全部经完备 GEN-CLOSURE 判定，0 闭合。三段管线在全空间端到端正确、sound、廉价。

## 性能

122 s / 100k（生成 12 s + 判定 110 s，单线程）。瓶颈是 ② chain_closure（对每个 multi-N 对算残基）；
③ GEN-CLOSURE 因残余 concordant 集极小而廉价。1M 阶段建议用 numpy 向量化生成 + 多进程判定。
