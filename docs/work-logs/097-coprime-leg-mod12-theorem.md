# wl097 — 互素腿 concordant N 的 mod-12 定理（闭合障碍的模骨架）

## 触发

承 wl096：在刻画"互素 pair 在图里的角色"时发现一条经验律——**互素 (A,B) 的每个 concordant N
都被 12 整除**，因此互素对里**永远没有互素的 N 对**。用户要求：先在大范围**推广验证**（不先证），
确认成立后再**证成定理**（推 mod 9 / mod 16 闭式）。本 worklog 完成证明。

## 推广验证（wl096，先于证明）

`scripts/multi_n/coprime_N_div12_check.py` 在三套独立数据上测「互素 ⟹ 12|N」，**零例外**：

| 来源 | 互素 multi-N pair | concordant N 值 | 最大坐标 | N 非 12 倍数 | 互素 N 对 |
|---|---|---|---|---|---|
| catalog ≤100k | 10,333 | 20,876 | 99,968 | 0 | 0 |
| 7M partner-BFS 顶点 | 15,706 | 32,029 | 6,974,471 | 0 | 0 |
| 穷举扫描 hyp≤5M | 580,828 | 1,165,393 | 4,999,995 | 0 | 0 |

合计 >1.16×10⁶ 个 concordant N，全部 ≡ 0 (mod 12)。

## 定理与证明（MATH §8.5.1）

**定理**：`gcd(A,B)=1` 且 `N²+A²=□`、`N²+B²=□` ⟹ `12 | N`。

**证明**：
- **3|N**：平方数 mod 3 ∈ {0,1}。`N²+A²=□` ⟹ `3|N ∨ 3|A`（否则 1+1=2 非平方）；同理 `3|N ∨ 3|B`。
  若 `3∤N` 则 `3|A ∧ 3|B`，与互素矛盾。故 `3|N`。
- **4|N**：互素 ⟹ A,B 不全偶，取奇腿 C，`C²≡1 (mod 8)`。平方数 mod 8 ∈ {0,1,4}。`N²+C²=□`：
  N 奇 → 和≡2（✗）；N≡2(4) → 和≡5（✗）；仅 N≡0(4) → 和≡1（✓）。故 `4|N`。

合并得 `12|N`。∎

**互素必要（边界 = §8.6）**：两步都死用互素。反例：
- `(6,15)` gcd=3（皆 3 倍数）→ N=8，`3∤8`（8²+6²=10²，8²+15²=17²）——(a) 塌；
- `(8,20)` gcd=4（皆偶）→ N=15 奇（15²+8²=17²，15²+20²=25²）——(b) 塌。

这就是 A.9 §8.6 gcd-scaling 漏洞的算术根源：约化（互素）腿享有 12|N，非互素腿 `(gA',gB')` 不受
此律约束、约化后不可见。

## 推论：闭合的模障碍

互素腿下 `12|(Nᵢ±Nⱼ)` ⟹ GEN-CLOSURE 四关系（wl093）要成立必须 `12|(A+B)` 或 `12||A−B|`。
catalog 10,333 个互素对里仅 30 个满足 `12|(A+B)`，且这 30 个 closure 仍=0 ⟹ **mod 12 一步排除
约 99.7% 互素对**。这是 `chain_closure_sieve` 模筛的显式闭式骨架，把"互素腿 GEN-CLOSURE 判定器
跑到 max_hyp=2000 无命中"往"闭式无反例"推进了一步（剩余的 thin set + 非互素腿仍开放）。
