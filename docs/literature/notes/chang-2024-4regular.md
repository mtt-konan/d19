## 与 d19 的关系

**澄清版本歧义**：
- **Chang 这篇是 graph drawing 版本**：每个 planar graph 是否能整数边长直线画法
- **d19 是 number theory 版本**：是否存在 rational distance 5-point set with chain structure

但 Chang 在论文末尾问的"five-point sets at rational distance"问题，**本质上就是 d19 项目在攻击的问题**。
