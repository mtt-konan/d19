# Greenfeld, Iliopoulou, Peluse 2024 — "On integer distance sets"

**arXiv**: 2401.10821 (v3 with HTML)
**优先级**：⭐ 提供 d19 问题的"广义版"密度定理
**状态**：abstract 读完

---

## 一句话总结

证明任何 integer distance set in $\mathbb{R}^2$（任意两点距离都是整数的点集）要么**极稀疏**（点数随直径几乎不增长），要么除少数 outlier 外都共线 / 共圆。

## 与 d19 的关系

**间接相关**：
- d19 的 4-chain 是"5 个点两两距离都为整数（更准确：5 个点之间的特定 6-7 个距离为整数）"
- Greenfeld 等的定理覆盖任意 integer distance set
- 他们的方法用**组合 + 调和分析**，跟 d19 用的 elliptic curve 方法**完全垂直**

## 对 d19 的启发

如果某天 d19 真找到了一个 4-chain，根据 Greenfeld 等的定理，这 5 个点必然共线或共圆——但 d19 的 4-chain 几何上既不共线也不共圆（4 个三角形构成一个正方形 + 内部 1 点）。

**这给 d19 提供了一个间接的"非存在性方向"**：能否构造 Greenfeld-Iliopoulou-Peluse 类型的密度论证，把 d19 的 5 点配置直接排除？

需要更多研究确认这是不是个 viable 角度。
